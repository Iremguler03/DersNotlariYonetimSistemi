# Ders Notları Yönetim Sistemi

## Proje Amacı

Bu proje kullanıcıların ders notlarını yönetebileceği bir Full Stack web uygulamasıdır.

Kullanıcılar:

- Kayıt olabilir
- Giriş yapabilir
- Ders notu ekleyebilir
- Dosya yükleyebilir
- Notlarını güncelleyebilir
- Soft delete ile silebilir
- Archive bölümünden kalıcı silebilir

## Kullanılan Teknolojiler

Frontend
- React.js
- Axios
- CSS

Backend
- ASP.NET Core Web API
- Entity Framework Core
- JWT Authentication

Database
- Microsoft SQL Server

## Özellikler

- JWT Authentication
- Dosya Upload
- Soft Delete
- Hard Delete
- Archive Sistemi
- REST API
